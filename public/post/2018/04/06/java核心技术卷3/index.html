  • 创建数组,都初始化为0;创建默认Boolean数组,都初始化为false;创建对象数组,都默认为null(比如String).

  • for each循环以及数组拷贝(一般用于扩大数组长度)

    int[] a =new int[10];
for (int element:a){
    element=1;
}
System.out.println(Arrays.toString(a));
for (int i=0;i<10;i++){
    a[i]=1;
}
System.out.println(Arrays.toString(a));
int[] b = new int[5];
b=Arrays.copyOf(a,8);
System.out.println(Arrays.toString(b));
