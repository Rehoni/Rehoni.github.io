Typora配置Github图床

rehoni / 2024-06-06

1、github

创建 github 私有仓库

生成一个 token 用于 PicGo 操作你的仓库：

访问：https://github.com/settings/tokens

然后点击 Generate new token。

2、picgo/picList

图床，选择“Github”作为图床服务。

在 Github 设置页面中，填写你的 Github 仓库信息，包括仓库名称、分支、文件夹等。

确保仓库是公开可访问的，并且已经创建了用于存储图片的文件夹。

可以启用 CDN 加速：https://cdn.jsdelivr.net/gh/user/repo@version/file

picList 可以开启删除云端功能，记得在设置中配置代理加速

3、typora

插入图片时规则——上传图片，并勾选对本地位置的图片应用上述规则

上传服务勾选 PicGo 或者 PicList，选中安装路径即可。

typora 如果显示图片比较慢，可以开启 typora 代理：Launch Arguments (Windows / Linux)
